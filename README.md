# VidJot Tutorial - Node.Js

Ideas for Youtube videos - alist app

## Installation

Install all dependencies

```
npm install
```

## Set up
If you are in local mode use:

```
npm run dev
```

To restart on save, install nodemon globally:
```
npm install -g nodemon
```
and then run:
```
nodemon
```


## Built With

* Node.Js
* Express.Js
* MongoDB
* Bootstrap
